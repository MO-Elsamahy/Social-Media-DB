## Features
- **User Management**: Handle user profiles, roles, and settings.
- **Content Management**: Create, update, and manage posts, comments, and hashtags.
- **Interactions**: Like, share, react, and bookmark posts.
- **Messaging**: Send direct messages and notifications.
- **Analytics**: Track user activity, post performance, and engagement.
- **Security**: Manage login attempts, password history, and blocked users.
- **Advertising**: Create and track ad campaigns and impressions.
- **Events**: Organize events and manage attendee RSVPs.
- **Support**: Handle support tickets and replies.

---

## Database Schema
The database is organized into the following schemas:

### **UserManagement**
- `Users`: Stores user profiles.
- `UserFollows`: Tracks follower relationships.
- `UserSettings`: Manages user preferences.
- `UserRoles`: Defines user roles.
- `UserRoleAssignments`: Assigns roles to users.

### **Content**
- `Posts`: Stores user posts.
- `PostMedia`: Manages media attached to posts.
- `Comments`: Stores comments on posts.
- `Hashtags`: Tracks hashtags used in posts.
- `PostHashtags`: Links posts to hashtags.

### **Interaction**
- `Likes`: Tracks likes on posts.
- `Shares`: Tracks shares of posts.
- `Reactions`: Tracks reactions (e.g., like, love) on posts.
- `Bookmarks`: Tracks bookmarked posts.

### **Messaging**
- `Messages`: Stores direct messages between users.
- `Notifications`: Manages user notifications.
- `MessageAttachments`: Stores attachments in messages.

### **Analytics**
- `UserActivity`: Logs user activities.
- `PostAnalytics`: Tracks post performance (views, likes, shares, comments).
- `UserEngagement`: Calculates user engagement scores.

### **Security**
- `LoginAttempts`: Logs user login attempts.
- `PasswordHistory`: Stores password change history.
- `BlockedUsers`: Tracks blocked users.

### **Settings**
- `PrivacySettings`: Manages user privacy preferences.
- `NotificationSettings`: Manages user notification preferences.

### **Advertising**
- `AdCampaigns`: Manages advertising campaigns.
- `Ads`: Stores individual ads.
- `AdImpressions`: Tracks ad impressions by users.

### **Events**
- `Events`: Stores event details.
- `EventAttendees`: Tracks event attendees and RSVPs.

### **Support**
- `SupportTickets`: Manages support tickets.
- `TicketReplies`: Stores replies to support tickets.
